# Release checklist

- [ ] `python scripts/verify_release.py` passes.
- [ ] `pytest -q` passes.
- [ ] Author names, affiliations, and ORCIDs are confirmed.
- [ ] CRediT assignments are approved by all authors.
- [ ] MIT license is approved by all code owners.
- [ ] `.zenodo.json` metadata are final.
- [ ] `CITATION.cff` metadata are final.
- [ ] The invalid draft dataset-overview figure is not included.
- [ ] GitHub repository is public and contains no secrets or raw restricted data.
- [ ] Full Git commit SHA is recorded.
- [ ] Zenodo integration is enabled before release publication.
- [ ] GitHub release `v3.0.0` is published from the verified commit.
- [ ] Zenodo version DOI and concept DOI are recorded.
- [ ] Manuscript Data and Code Availability statement is updated.
